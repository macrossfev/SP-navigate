## 推荐算法：地理围栏法

### 算法原理

```
针对长寿区沿长江东西向分布的特点：

1. 按经度排序所有点位（从西到东）
2. 按顺序切片分组（每组约 12-15 点）
3. 组间边界 = 经度线，天然不重叠
```

### 为什么选择这个算法？

| 算法 | 组间边界 | 临近聚合 | 点数控制 | 实现难度 |
|------|---------|---------|---------|---------|
| TSP | ❌ 无边界 | ❌ 全局路径 | ✅ 可控制 | 中 |
| K-Means | ❌ 重叠 | ⚠️ 一般 | ⚠️ 不精确 | 中 |
| DBSCAN | ❌ 重叠 | ✅ 好 | ❌ 不可控 | 中 |
| **地理围栏** | ✅ **经度线** | ✅ **好** | ✅ **精确** | **低** |

## 算法实现代码

```python
def geographic_fence_clustering(points, target_groups=5, max_points=15):
    """
    地理围栏法 - 按经度切片
    
    针对长寿区沿长江东西向分布的特点
    组间边界 = 经度线，天然不重叠
    """
    n = len(points)
    
    # 按经度排序（从西到东）
    sorted_indices = sorted(range(n), key=lambda i: points[i]['lng'])
    
    # 计算每组点数
    points_per_group = max(1, (n + target_groups - 1) // target_groups)
    
    # 切片分组
    clusters = []
    for i in range(0, n, points_per_group):
        cluster = sorted_indices[i:i + points_per_group]
        if cluster:
            clusters.append(cluster)
    
    return clusters
```
